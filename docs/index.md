# Planejamento Anual de Compras (PAC) - Fundação Hospitalar do Estado de Minas Gerais

Esta página reúne as principais informações a respeito do **Planejamento Anual de Compras (PAC)** da Fundação Hospitalar do Estado de Minas Gerais – FHEMIG. Em diferentes sessões, você poderá encontrar:

1. Base de conhecimento para execução do planejamento.
2. Cronograma de execução das iniciativas.
3. Dados consolidados do Planejamento Anual de Compras.
4. Histórico de anos anteriores.
