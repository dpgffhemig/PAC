# PAC 2026

## Visão geral

A janela para realização do **PAC 2026** ocorreu entre **07/10/2025** e **19/12/2025**.  
No exercício de 2026, ocorrerão os **ciclos de revisão**, sendo o primeiro programado para
o período de **02/03/2026 a 20/03/2026**. Confira o quadro abaixo com o cronograma completo.

## Cronograma

| ETAPA | ATIVIDADES                                                                 | PRAZO                 |
| :---: | -------------------------------------------------------------------------- | --------------------- |
|  1    | Planejamento de solicitações                                               | 24/11/2025            |
|  2    | Aprovação do planejamento de solicitações (*etapa realizada pelos Ordenadores de Despesas*) | 22/12/2025            |
|  3    | Planejamento de processos de compra                                        | 30/12/2025            |
|  4    | Vinculação dos itens aos processos de compra                               | 30/12/2025            |
|  5    | 1ª revisão                                                                 | De 02/03 a 20/03/2026 |
|  6    | 2ª revisão                                                                 | De 22/06 a 10/07/2026 |

!!! note "Observação"
    Datas sujeitas a ajustes comunicados pela coordenação do PAC.
